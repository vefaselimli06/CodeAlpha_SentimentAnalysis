# Sentiment Analysis of Customer Reviews
## Project Overview
This project was completed as part of the **CodeAlpha Data Analytics Internship (Task 4)**. The objective was to analyze customer reviews, identify sentiments, and visualize the results using Python.
## Tools & Libraries
- Python
- Pandas
- Matplotlib
- Seaborn
- WordCloud
## Dataset
The project uses a customer review dataset containing positive and negative sentiments.
## Visualizations Included
- Sentiment Distribution
- Review Length Distribution
- Positive & Negative Word Clouds
- Sentiment Percentage
## Key Insights
- The dataset contains a balanced distribution of positive and negative reviews.
- Most customer reviews are relatively short.
- Word clouds reveal the most frequently used words in positive and negative feedback.
- Sentiment analysis helps identify customer opinions and overall satisfaction.
